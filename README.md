# sholat-linux

CLI jadwal sholat akurat untuk Linux.
Cocok untuk KDE Panel, GNOME, Waybar, i3, dll.

## Fitur
- Akurat (berbasis kota resmi MyQuran)
- Setup 1x, lalu tinggal pakai
- Output 1 baris (panel friendly)
- Cross-distro
- Aman jika offline

## Dependency
- bash
- curl
- jq
